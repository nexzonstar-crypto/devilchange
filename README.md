# Portfolio Project - AI Tools Setup

## Tools Installed
- Cursor IDE (from https://cursor.com/)
- Claude Code add-on (installed via Extensions)
- Codex add-on (installed via Extensions)

## Steps Completed
1. Installed Cursor IDE on Windows 10
2. Added and logged into Claude Code add-on
3. Added and logged into Codex add-on
4. Created a public GitHub repository
5. Opened the repository in Cursor
6. Created README.md with this documentation
7. Committed and pushed to GitHub

## Issues Encountered and Solutions
- **Issue**: Couldn't find Claude Code in Extensions  
  **Solution**: Typed the exact name "Claude Code" and checked the official publisher

- **Issue**: Git not recognized when pushing  
  **Solution**: Installed Git for Windows from https://git-scm.com/ and restarted Cursor

- **Issue**: Didn't know how to create a GitHub repo  
  **Solution**: Watched YouTube tutorial "how to create a GitHub repository" [web:1]

- **Issue**: Unsure how to push code  
  **Solution**: Used `git add .`, `git commit -m "initial commit"`, `git push origin main` [web:2][web:6]
